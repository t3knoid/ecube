## 2. Home Page Structure

### Hero section

Suggested headline:

Secure evidence export without giving operators direct filesystem control.

Suggested subheading:

ECUBE moves approved case data from network shares to encrypted USB drives through a governed, audited, browser-based workflow built for legal, forensic, and compliance teams.

Primary CTA: Request a Demo
Secondary CTA: View Appliance Options

### Problem and solution section

Show the contrast between manual copying and the ECUBE approach.

Suggested message points:

- Manual copy workflows depend on workstation access and operator discretion.
- ECUBE introduces role-aware controls, structured audit logs, and repeatable export steps.
- The result is faster delivery with better defensibility and less operational risk.

### Feature grid

Highlight these feature cards:

- Audit trail and logging
- Role-based access control
- Simple task-focused interface
- Multi-drive export workflow
- REST API and webhooks
- Compliance and chain-of-custody support
- Themes and browser-based UX

### Screenshot strip

Use real product screenshots from the Dashboard, Jobs, Drives, Mounts, and Audit pages.

### Trust section

Suggested bullets:

- Designed for evidence handling and repeatable exports
- Supports NFS and SMB evidence sources
- Integrates with enterprise authentication and automation workflows
- Available as software or as a turnkey hardware appliance
- Purpose-built appliance option uses dedicated USB 3.1 controllers for optimal throughput

## 3. Software Page

Purpose: explain ECUBE as a secure evidence export platform.

Recommended sections:

- What the software does
- Feature walkthrough
- API and webhook integration
- Deployment models
- UI themes and branding support
- Screenshot-led workflow overview

Suggested message:

ECUBE gives operations teams a controlled way to export evidence with visibility, auditability, automation support, and a simple interface that stays focused on each user’s role and task.

## 4. Appliances Page

Purpose: explain the turnkey offering and why validated hardware matters.

Recommended sections:

- Why a turnkey appliance reduces rollout risk
- Standard, Professional, and Enterprise tiers
- Dedicated USB 3.1 controllers and why they matter for sustained write speed
- USB concurrency and controller isolation
- Secure deployment options for air-gapped or enterprise environments
- Request a sizing discussion

Suggested message:

ECUBE appliances combine validated hardware and secure workflow software into a dedicated export station that is ready for production use, with dedicated USB 3.1 controller hardware selected for optimal multi-drive export speed.

## 5. Security and Compliance Page

Purpose: address buyer objections directly.

Recommended proof points:

- RBAC separates admin, manager, processor, and auditor responsibilities.
- Audit logs capture authentication, copy, verification, and device events.
- Project isolation helps prevent cross-project contamination.
- Verification and manifest generation support defensibility and review.
- Browser-based operations reduce direct handling of source and destination filesystems.

## 6. Demo Page

Purpose: convert interested visitors into a guided product evaluation.

Recommended content:

- short explanation of what the demo shows;
- note that the demo uses sanitized sample data;
- request-access or shared-evaluation flow;
- screenshot preview and expected demo scenarios.

## 7. Simple Visual Design Direction

Use a clean, modern, high-trust design with restrained color use, generous spacing, and screenshot-first storytelling.

Recommended style choices:

- dark navy or charcoal for headings and trust sections;
- crisp white or light neutral backgrounds for content blocks;
- one accent color for CTAs and status highlights;
- simple sans-serif typography;
- iconography that reinforces security, auditability, hardware, and workflow.

## 8. Navigation and Footer

Top navigation should include Home, Software, Appliances, Demo, and Contact.

Footer should include:

- short product summary;
- links to Security and Compliance, Contact, and Demo;
- copyright and company details;
- optional note that ECUBE is available as software and appliance bundles.
