## Purpose

This document will serve as the end-user manual for the ECUBE web interface. It will cover day-to-day workflows as performed through the graphical UI, including:

- Logging in and navigating the dashboard
- Viewing and managing USB drives
- Creating, starting, and monitoring export jobs
- Reviewing copy results, file manifests, and hash verification
- Browsing and exporting audit logs
- Managing network mount sources

## Scope

This manual is intended for operators who interact with ECUBE through the web-based frontend. For backend administration tasks (service configuration, OS-level user management, troubleshooting, and maintenance), see the [Administration Guide](06-administration-guide.md). For direct API usage, see the [API Quick Reference](08-api-quick-reference.md).
