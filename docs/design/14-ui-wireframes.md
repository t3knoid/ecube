**Header bar** (UC-2.3, UC-2.4, UC-2.5):

| Area | Position | Contents |
|------|----------|----------|
| **Logo placeholder** | Left | Reserved space for organization/product logo image (configurable) |
| **App name** | Left (beside logo) | "ECUBE" label |
| **User info** | Right | Logged-in username + role badge(s) + dropdown menu |
| **Token expiry** | Right | Countdown timer showing remaining session time (вҸұ) |
| **Log Out button** | Far right | Explicit logout action вҖ” ends session and redirects to login screen |

**Sidebar navigation** вҖ” items visible based on role:
- Dashboard, Drives, Mounts, Jobs, Audit вҖ” visible to all roles
- Users вҖ” visible to admin only (UC-3.*)
- System вҖ” visible to all (introspection is read-only)

**Footer status bar** (UC-8.1, UC-8.2):
- Application version
- Database connection indicator (green/red dot)
- Active job count
