# 8. Programming Language & Framework Requirements

## 8.1 ECUBE System Layer

- Python 3.11+
- FastAPI
- SQLAlchemy + Alembic
- Celery/RQ for background tasks
- Linux OS

## 8.2 ECUBE UI

- React/Vue or server-rendered templates
- HTTPS-only communication

## 8.3 Database

- PostgreSQL
- Only accessible by ECUBE System Layer
